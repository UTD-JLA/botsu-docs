# Introduction

This page serves as a brief introduction to Botsu, and provides information on what to expect from this guide.

## This Guide
This guide is intended to help you get started with Botsu, and to help you understand how to use Botsu's features.

For information useful for more advanced usage, see the [Reference Guide](/reference/).

For information on self-hosting, see the [Self-Hosting Guide](/self-hosting/).  

## What is Botsu?
Botsu is a Discord bot for logging time spent on Japanese media.
Time is logged with the `/log` family of commands, which can be accessed across
all servers that Botsu is in, as well as in DMs with Botsu.

### Use Case
Botsu is great for logging time spent on Japanese media, such as anime, manga, light novels, YouTube videos, etc.
Immersion is an important part of learning a language, and Botsu helps you keep track of your immersion time, as well as
providing you the opportunity to share the media you've consumed with others.

### Comparing TMW's Bot
While TheMoeWay also has a bot for logging immersion time, Botsu has a few key differences:
- Botsu does not require each server to host their own instance.
- Logs are not tied to a specific server, and can be viewed from any server that Botsu is in.
- Immersion is not measured in "points", but rather in time spent.
